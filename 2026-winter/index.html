対象者

SAGA IT Community Day は、佐賀県内で IT やデジタルに興味のある方なら、どなたでも参加できる学びのイベントです。

次のような方におすすめです。

  • 佐賀県内で IT・デジタルの仕事に関わる方
    （開発・デザイン・AI・システム運用、ローコード／ノーコード、データ分析・業務改善など）
  • これから IT を学びたい県内の学生・社会人
    （高校・高専・大学の学生、独学中の方、プログラミングが好きな方）
  • 県内企業・団体・自治体・教育機関でデジタル活用に取り組みたい方
    （業務改善、データ活用、IT 導入、Google Workspace や GAS、Power Platform をもっと使いこなしたい担当者 など）
  • 地域の学習会・創作活動に関わる方
    （プログラミング教室、勉強会 など）
  • プログラミング／AI／IoT などを趣味や副業として始めてみたい個人の方
  • 日常の仕事や地域活動で「IT に課題がある」「同じ悩みを持つ人とつながりたい」と感じている方
  • 「コミュニティはちょっと怖い」「人脈づくりは苦手だけど、ゆるく関わってみたい」という方
    ／「IT コミュニティってどんなところ？」と雰囲気を知りたい方

佐賀県内で「ちょっと面白そうだな」と思ったら、ぜひお越しください。

年齢・経験・職業は問いません。

参加すると

  • ITやデジタル技術について、初心者でもわかりやすく学べる
    難しい話だけではなく、身近な視点で理解できる内容を用意しています。
  • 佐賀県内のITの取り組みや学びの環境がよくわかる
    地域でどんな活動が行われているか、どこから参加できるかを把握できます。
  • 「まずはやってみる」の一歩が踏み出せる
    小さな発表や共有も歓迎される場で、学んだことを誰かに伝える体験ができます。
  • 100回参加するより、1回登壇する価値に出会える
    完成した成果でなくても、途中経過や小さな工夫が誰かの学びを生みます。
  • 初心者でも安心して参加できる雰囲気
    気軽に相談できる環境で、ITに不慣れでも安心して過ごせます。

Google Apps Script（GAS）を活用した業務効率化や自動化について学ぶコミュニティです。初心者から上級者まで、GASの可能性を一緒に探求していきます。

GABAITech

佐賀県のIT技術者コミュニティです。最新技術の情報共有や勉強会を通じて、地域のIT技術者のスキルアップと交流を促進しています。

Webサイト Connpass

HAGAKURE PROGRAMMING塾

佐賀県内で世代・職業を超えて同志とともに楽しく成長するプログラミング塾です。みんな塾生、みんな先生。アウトプットすることで「自分の学び」を「みんなの学び」へ。

Webサイト 公式サイト X Facebook

JAWS-UG佐賀

Amazon Web Services（AWS）のユーザグループ、JAWS-UGの佐賀支部です。AWSに興味がある方、初心者の方も大歓迎。佐賀県内外からの参加者を歓迎しています。

Webサイト Google Sites Connpass Facebook

JBUG佐賀

Japan Backlog User Groupの佐賀支部です。Backlogに関する情報交換や勉強会を開催し、佐賀県内の開発者のスキル向上とコミュニティ形成を支援しています。

Webサイト Connpass

ハックツハッカソンコミュニティ

ハッカソンを通じて技術者同士の交流とスキルアップを図るコミュニティです。様々なレベルの参加者が集まり、協力してプロダクトを開発する場を提供しています。

Webサイト Doorkeeper

ローコードツール勉強会

主にマイクロソフトのローコードツールであるPower Platformのうち、Power Apps、Power Automate、Power BIを主に勉強する集まりです。基本はオンラインでの勉強会ですが、MAIC 佐賀での勉強会開催も予定しています。

FAQ

SAGA IT Community Day実行委員会とはなんですか？

表示されている各コミュニティから有志を募って本イベントを企画・運営しています。

参加に必要なスキルレベルはありますか？

いいえ、初心者から上級者まで、すべてのレベルの方が参加できます。

ノートPCは必要ですか？

セッションによっては必要な場合があります。貸出の予定はありません。詳細はスケジュールをご確認ください。

参加費以外に費用は掛かりますか？

懇親会に参加される方は別途掛かります。

自家用車で行ってもいいですか？

はい、可能です。LIGHTHOUSEには無料駐車場はございませんが、提携のコインパーキング「唐人南パーキング」をご利用いただけます。

  • 料金：1時間100円、400円で12時間まで駐車可能
  • 会場から徒歩2分
  • 最大400台駐車可能

詳細はLIGHTHOUSEの駐車場案内をご確認ください

入退場自由とはどういうことですか？

来場する時間、お帰りになる時間は自由ということです。ただしハンズオンなど途中参加ではゴールまで行けない可能性がありますのでご注意ください。

AWS Builders Cardsとはなんですか？

AWSドイツのゲーム業界に携わるエンジニアが制作したサービスやアーキテクチャを楽しみながら学べるトレーディングカードゲームです。AWSのサービスを知らない方でも、ゲームを通してAWSのサービスやアーキテクチャを学ぶことができます。AWSのサービスやアーキテクチャを知らない方は、この機会にAWSのサービスやアーキテクチャを学んでみてください。

対象年齢 12才以上
1チームのプレーヤー 2-4名
1回のプレイ時間 20分ほど

(参考)楽しみながら学べる AWS BuilderCards の遊び方、そして日本語化に込めた思い

プロジェクトテーマパークとはなんですか？

株式会社ヌーラボが制作した、テーマパークを題材にしたプロジェクト管理を学べるボードゲームです。それぞれに役割が開始前に与えられ、チームでテーマパークを建設します。

対象年齢 13才以上
1チームのプレーヤー 3-5名
1回のプレイ時間 30-40分ほど
